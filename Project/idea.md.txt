ไอเดียเเอพ :
เเอพทำฟอร์มสำหรับคอมมูเเละโรลเพลย์
- ทางหัวโรลหรือสตาฟสามารถสร้างเเบบฟอร์มให้ตอบ
- ทางผู้เล่นสามารถกรอกข้อมูลลงไปในนั้นได้เลยโดยจะมีการรวมข้อมูของผู้เล่นเอาไว้ให้ฝั่งของหัวเเละสตาฟสามารถเข้าไปดูได้
-  สามารถบันทึกข้อมูลการเเก้ไขฟอร์มได้ตลอดเวลา

ระบบคร่าวๆ
- สร้างต้นฟอร์ม ( หัว+สตาฟ )
- เขียนฟอร์ม ( ผู้เล่น )
- ระบบบันทึกประวัติการแก้ไข ( หัว+สตาฟ ) ( ผู้เล่น )
- ค้นหา / กรองข้อมูล ( หัว+สตาฟ ) ( ผู้เล่น )
